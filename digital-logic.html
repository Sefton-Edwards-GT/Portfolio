Digital Logic & FSM Circuit Modeling

Introduction

In this project, I designed and simulated two digital logic systems using Multisim: a gate-restricted combinational circuit and a finite state machine (FSM). The combinational circuit was built using only NAND gates and inverting decoders, while the FSM was designed to perform defined state transitions based on input and clock signals. Both simulations demonstrated combinational logic's role in creating predictable digital behavior.

Problem

The project had two main goals. First, to design a combinational circuit using only restricted logic gates that performed specific Boolean functions. Second, to design a three-state FSM that accurately transitioned between states based on input signals and the current stored state. Both circuits needed to operate reliably and match theoretical expectations in simulation.

Action

For the combinational component, I derived Boolean expressions for the desired logic functions and implemented them using NAND gates and again with NOR and NOT gates, being sure to convert into equivalent gates. Each signal path was labeled to ensure clarity during simulation, and the outputs were tested for correctness under all input conditions. For the FSM component, I mapped a state diagram and created a state transition table for three states. Two D flip-flops were used to store the current state, and a 3-to-8 decoder with inverted outputs generated the next-state logic. The clock, input, set, and reset signals were connected to interactive digital constants, and the FSM was advanced manually to verify each state transition against the theoretical model.

Process

I began by mapping the state transition diagram and truth table, defining three distinct states represented by two binary outputs (S1, S0). Using Multisim’s Misc-Dig TIL library, I placed and connected the core components: a 3-to-8 decoder with inverted outputs, two edge-triggered D flip-flops for state storage, and a network of NAND gates for combinational logic. Each decoder output represented a unique state condition, and NAND gates were configured both as inverters and logical conjunctions to generate next-state equations. I synchronized both flip-flops with a shared clock input, using interactive digital constants for the clock, set, reset, and input signals. I then manually toggled the clock to advance the FSM one state at a time, verifying each transition matched the theoretical model. Each connection was labeled for signal clarity, and iterative simulations were performed to confirm timing accuracy and logical consistency.

Result

The combinational circuit correctly implemented the desired logic functions using gate restrictions. The FSM successfully cycled through all three states according to the state table, with outputs changing correctly on positive clock edges. Both components operated reliably, confirming the accuracy of the designs in simulation.

Reflection

This project improved my understanding of the relationship between combinational and sequential logic. Building the circuits in two separate modules helped clarify how individual logical functions contribute to overall system behavior. Working with restricted gates reinforced Boolean simplification and careful labeling, while the FSM simulation highlighted the importance of clock timing and initialization. Overall, the project provided practical experience in moving from theoretical design to functional digital systems.
