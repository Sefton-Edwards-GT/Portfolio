Motion Control & Systems Integration

Engineering Team Member — Quanser Self-Driving Car Competition

Introduction

Selected from my research lab to join Louisiana State University’s Quanser Self-Driving Car Competition team, I contributed to the development and testing of motion control algorithms for an autonomous vehicle. Working within an eight-member interdisciplinary team of undergraduate and graduate students, I focused on the motion control subsystem. I developed Python-based navigation and control logic to guide the car through straight paths, turns, and roundabouts. The project emphasized precision control, system reliability, and effective collaboration across teams.

Problem

The core challenge was achieving smooth and stable motion through multi-waypoint routes while maintaining a steady velocity of 0.5 m/s. Our system needed to handle directional transitions and turning maneuvers accurately while synchronizing motion control with the computer vision subsystem for route detection and obstacle handling.

Action

I refined controller parameters through repeated tests to ensure stability, responsiveness, and consistent navigation. Using iterative debugging and performance analysis, I optimized control smoothness, leading to reliable path execution. Each ~30-second test run demonstrated the integration between motion and vision systems while maintaining precise speed control and steering behavior.

Process

Following the V-shaped development model, I began by writing and testing Python programs to control forward, reverse, left, and right motion. I collaborated closely with the vision sub-team to integrate perception data with motion control, using Box cloud folders for shared code access and Microsoft Teams for coordination. I also developed real-time dashboard tools that tracked speed, steering angle, and trajectory performance, capturing over 100 data points per second for analysis and optimization.

Result

Our system successfully completed all four routes at 0.5 m/s, achieving smooth transitions and accurate path following through turns and roundabouts. The dashboard visualization enabled real-time monitoring and data-driven refinement of system performance. The final implementation met objectives and operated with consistent precision during testing and demonstrations.

Reflection

Being selected to represent my lab on the LSU Quanser Self-Driving Car team was an invaluable experience that strengthened my skills in motion control, Python programming, and interdisciplinary teamwork. I learned how to integrate control logic with perception systems, analyze real-time data, and apply structured development processes which are skills that continue to guide my approach to autonomous system design and control engineering.
